## 프로젝트 실행 방법

- node 21.6.2 버전 이상
- 프로젝트 root 경로에서 npm install 로 패키지 다운로드
- npm start 실행 (localhost:3000)
- Prettier 사용 (vscode extension)

## 사용 라이브러리

- react 18.3.0
- typescript 4.9.5
- ant-design (UI 라이브러리)
  - 프로젝트 내 UI 개발을 위해 사용
  - message alert, button 등을 위해 사용
- tailwind (CSS 라이브러리)
- axios (REST API 통신 라이브러리)
- recoil (상태 전역 관리를 위해 사용)

## 디렉토리 구조

- /components : 컴포넌트
- /pages : UI 페이지
- /service : store, api 호출, atom 관리
- /type : tpyescript type 폴더
- /util : 공통 함수 폴더
- 실무에서는 각 폴더에 프로젝트 메뉴나, 도메인에 따라 세분화 하여 관리합니다.

## 설계 방식

- 달력은 2차원배열로 생성하였습니다.
- api는 처음 1회 호출 후 공고와 직무 데이터는 recoil로 전역 관리하였습니다.
- 홈(calendar) 페이지와 직무 모달, 상세정보 모달 컴포넌트로 구성되어 있습니다.

## 주요 기능

1. 달력에서 공고를 확인할 수 있습니다.
2. 날짜 좌우 아이콘 통해 이전달, 다음달로 이동이 가능합니다.
3. 공고는 시작되는 공고, 끝나는 공고로 구분됩니다.
4. 공고를 확인하면 상세 모달이 띄워지고 읽음 처리 됩니다.
5. 상단 직무 필터를 통해 공고를 필터링할 수 있습니다.
6. 직무 필터 우하단 초기화 버튼으로 필터링을 초기화할 수 있습니다.

### 미비한 기능

1. 직무 계층 중 최상위 계층(1depth) 체크박스 클릭 시 최하위 계층(3depth) 필터링을 구현하지 못했습니다.
2. 공고 상세 모달 이전/이후 공고에서 필터링을 구현하지 못했습니다.
